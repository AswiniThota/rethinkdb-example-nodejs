Introduction

This little example illustrates Benford's law using Twitter's streaming API.
The number of occurrences of each significant digit is computed and updated in real time.

How it works

This example is composed of two parts:

  • A Node.js script listen to Twitter's sample stream, extract the first significant digits of the numbers and save it in RethinkDB.
  • A Node.js server listen to any database changes, and broadcast it to any client connected using Socket.io

You can find the code on GitHub, in the node.js examples repository
